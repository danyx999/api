- **Graphical Calculator**

- **Infoprog**
  - Project for a competition

  - Used TDD
  - CLI-based UI
- **Register**
  - Assignment for practising
  - CLI-based UI
  - Can add custumers and mark them as completed
  - Can open and close registers
  - Used TDD
  - In progress
